# Laudable l10n HTML element specification

The aim is to implement gettext-like functionality within the scope of this theme

### Basic message
<l-10n>[MESSAGE]</l-10n>
[MESSAGE] will be localised if translation is available

### Message with context
<l-10n context="[CONTEXT]">[MESSAGE]</l-10n>

### Message within a HTML attribute
<l-10n attr="placeholder">
    <input type="text" placeholder="[MESSAGE]" />
</l-10n>
